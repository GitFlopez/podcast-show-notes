# Prompt 2: Episode Description (Apple Podcasts / Spotify)

## When to Use
Use this for the short description that appears in Apple Podcasts, Spotify, and other podcast directories. This is what shows up in search results — it's your hook.

---

## The Prompt

```
You are a podcast marketing specialist who writes episode descriptions that rank in Apple Podcasts and Spotify search AND make people click play.

Write a podcast directory description for this episode:

**Episode Details:**
- Podcast name: [YOUR PODCAST NAME]
- Episode title: [EPISODE TITLE]
- Guest name and expertise: [NAME — what they're known for] (or "Solo episode" if no guest)
- Episode topic in one sentence: [WHAT THIS EPISODE IS ABOUT]
- Main keywords your listeners search: [2-3 SEARCH TERMS YOUR AUDIENCE USES]

**Top 3 things listeners will learn:**
1. [INSIGHT OR TAKEAWAY #1]
2. [INSIGHT OR TAKEAWAY #2]
3. [INSIGHT OR TAKEAWAY #3]

---

Write the episode description following these rules:

**Structure:**
- First sentence (10-15 words): The most compelling hook. What transformation or insight does this episode deliver? Make it impossible to scroll past.
- Body (3-4 sentences): Expand on what they'll learn. Use natural language. Weave in 1-2 keywords without keyword stuffing.
- Closing line: Direct CTA — "Hit play now," "Listen now," or similar.

**Rules:**
- Total length: 150-200 words exactly
- Tone: Match the energy of a show that respects listener intelligence
- No clickbait — deliver on the promise in the description
- First 5 words are the most important — many apps truncate after that
- Include guest name + main keyword in first 2 sentences for directory SEO

Output the description as clean, ready-to-paste text with no extra formatting.
```

---

## Pro Tip
Paste this description into both the "description" field in your podcast host (Buzzsprout, Anchor, etc.) AND use it as the meta description for your show notes page. Double the SEO work with one prompt.
