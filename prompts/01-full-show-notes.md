# Prompt 1: Full Show Notes (SEO-Optimized)

## When to Use
Use this to generate the complete show notes page that lives on your website. Optimized for Google search and podcast listener experience.

---

## The Prompt

```
You are an expert podcast show notes writer who specializes in SEO-optimized content that ranks on Google and converts listeners to subscribers.

Write complete show notes for this podcast episode:

**Episode Details:**
- Podcast name: [YOUR PODCAST NAME]
- Episode number: [Ep. XX]
- Episode title: [EPISODE TITLE]
- Guest name and title: [NAME, TITLE at COMPANY] (or "Solo episode" if no guest)
- Episode length: [XX minutes]

**Episode Content:**
[PASTE TRANSCRIPT OR DETAILED BULLET POINTS OF WHAT WAS DISCUSSED]

**Resources mentioned:**
[LIST ANY TOOLS, BOOKS, WEBSITES, OR LINKS MENTIONED]

---

Write the show notes in this exact structure:

1. **Episode Summary** (150-200 words)
   - Hook sentence that makes someone want to listen
   - What the listener will learn
   - Why this episode matters right now

2. **In This Episode** (timestamped chapters)
   - Format: [00:00] Topic name
   - Include 6-10 chapters
   - Chapters should be specific, not generic

3. **Key Takeaways**
   - 5-7 bullet points
   - Each starts with an action verb
   - Specific and actionable, not vague

4. **Guest Bio** (skip if solo episode)
   - 2-3 sentences
   - Credentials that establish authority
   - What they're known for

5. **Resources & Links**
   - Everything mentioned in the episode
   - Format: [Resource Name](URL or "link in bio")

6. **Connect With [Guest Name / Host Name]**
   - Social handles
   - Website
   - Any free offers mentioned

Use H2 for section headers, H3 for sub-sections. Include the episode title and guest name naturally 2-3 times for SEO. Write in a direct, conversational tone that matches a podcast audience.
```

---

## Tips for Best Results

- The more detail you give in "Episode Content," the better the output
- Timestamps can be approximate — the AI will space them logically
- If you have a transcript, paste the full thing — longer input = better output
- Run the output through Grammarly for a final polish pass
